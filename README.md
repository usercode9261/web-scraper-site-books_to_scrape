# web-scraper-site-books_to_scrape
web scraper of the site "books.toscrape.com", this code extract 5 elements:
##elements extracted:##

#title:# extract the name of the book.
#rating:# extract the rating of the book, I created a filter to extract just books with the rating equal or above 3 stars(maximum of 5 stars).
#price:# extract the price of the book(without the symbol of euro, allowing interact with this element as a number and not a text).
#date:# put the date that the datas were collect.
#link#: extract the link of the book, allowing the user click on it and acess the page that the book is located.
